## Interactive Learning Environments

These GitHub Codespaces repositories provide hands-on learning experiences for bioinformatics and computational biology:

### Metagenomics Analysis and Shell Novice

**[Link to Codespace](https://github.com/dkbiocode/trainings){:target="_blank"}**

<img alt="Screenshot of shell novice" src="/assets/images/shell-novice-screenshot.png" width="50%">

**Older versions in separate environments:**

#### Metagenomics Analysis
**[codespace-metagenomics-analysis](https://github.com/dkbiocode/codespace-metagenomics-analysis){:target="_blank"}**

An interactive environment for learning metagenomics analysis workflows, including sequence processing, taxonomic classification, and microbial community analysis.

#### Metagenomics Shell Skills
**[codespace-metagenomics-shell](https://github.com/dkbiocode/codespace-metagenomics-shell){:target="_blank"}**

Learn essential shell scripting and command-line skills specifically tailored for metagenomics data processing and analysis.

#### Shell for Beginners
**[codespace-shell-novice](https://github.com/dkbiocode/codespace-shell-novice){:target="_blank"}**

A beginner-friendly introduction to the Unix shell, covering fundamental concepts and commands for working in command-line environments.

### CM515 Course materials

I served as course coordinator for the Cell and Molecular Biology graduate program. The curriculum was an introduction to bioinformatics, R, and computational approaches to handling modern data in biology.

In addition to coordinating guest lecturers from faculty to present their lab's research, I directed student instructors to present material from their research.
